##  Project Description
This project is an audit of the open-source software Git.  
It includes analysis of its origin, philosophy, licensing, Linux footprint, and ecosystem.

Additionally, 5 shell scripts are developed to demonstrate Linux and Bash scripting skills.

##  Shell Scripts

###  Script 1: System Identity Report
Displays system information such as:
- Kernel version
- User name
- Uptime
- OS details
